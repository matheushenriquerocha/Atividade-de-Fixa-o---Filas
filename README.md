# Atividade_Filas (Estrutura de Dados)

Este repositório contém uma série de exercícios desenvolvidos em Python para praticar a implementação e utilização de filas (queues) usando listas.
Os exercícios simulam situações reais como atendimento, impressão de documentos e processamento de tarefas em servidores.

Cada exercício explora uma aplicação prática da estrutura de dados Fila (FIFO – First In, First Out).
